# Auditoría de Inventario
Este proyecto es una solución tecnológica automatizada en Python para el control de existencias, desarrollada como la Evaluación Final (POA) del curso **Fundamentos de Programación (Código: 213022_560)** de la UNAD.

## Características
- **Enfoque Estructurado:** Implementación modular mediante funciones y estructuras de control.
- **Uso de Matrices:** Almacenamiento organizado de artículos (Código, Nombre, Stock Actual, Stock Mínimo).
- **Menú de Consola:** Opciones interactivas para registrar productos, visualizar el inventario y generar reportes automatizados de reabastecimiento con totales acumulados.
