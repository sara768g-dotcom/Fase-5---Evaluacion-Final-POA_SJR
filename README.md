# Auditoría de Inventario y Reabastecimiento

Este proyecto presenta una solución automatizada en Python para la gestión y auditoría de inventarios, correspondiente a la Evaluación Final (POA) del curso **Fundamentos de Programación (Código: 213022)** de la Universidad Nacional Abierta y a Distancia (UNAD).

## Características
- **Paradigma Estructurado:** Implementación limpia mediante funciones y estructuras de control.
- **Matriz de Datos:** Almacenamiento organizado de artículos (Código, Nombre, Stock Actual, Stock Mínimo).
- **Menú Interactivo:** Opciones para registrar artículos, visualizar el inventario completo y generar reportes analíticos de unidades a solicitar.
