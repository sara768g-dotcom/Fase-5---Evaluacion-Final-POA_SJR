# Auditoría de Inventario y Reabastecimiento

Este proyecto presenta una solución tecnológica automatizada en Python para la gestión y auditoría de inventarios, desarrollada como componente práctico para la Evaluación Final (POA) del curso **Fundamentos de Programación (Código: 213022)** de la Universidad Nacional Abierta y a Distancia (UNAD).

## Descripción del Proyecto

El programa está diseñado bajo un enfoque estructurado y orientado a la automatización del control de existencias en entornos empresariales. Permite almacenar de manera organizada la información de los artículos utilizando matrices (listas de listas) y evalúa mediante lógica condicional si los productos requieren reabastecimiento, calculando de forma exacta la cantidad de unidades a solicitar.

## Características Principales

- **Menú Interactivo:** Interfaz de consola dinámica para la navegación entre las funciones del sistema.
- **Registro de Artículos:** Permite la captura de datos ingresados por el usuario (Código, Nombre, Stock Actual y Stock Mínimo) para poblar la matriz de inventario.
- **Visualización Estructurada:** Muestra el inventario completo registrado en un formato de tabla limpio y ordenado.
- **Auditoría Automatizada:** Procesa la matriz utilizando estructuras repetitivas (`for`) e invoca un módulo independiente (función) para generar el informe analítico de reabastecimiento con sus respectivos totales acumulados.

## Requisitos de Ejecución

- Python 3.x instalado en el computador.
- Terminal o consola de comandos para la ejecución del script.
